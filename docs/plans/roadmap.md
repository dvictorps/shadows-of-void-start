# Item System Roadmap

## Unique Items (planned)

- New rarity tier above Epic: **Unique**
- Each unique item is a hand-crafted template with fixed, curated modifiers
- Unique modifiers can be exclusive — not found on any other item in the game
- Uniques define the build-around fantasy: powerful but narrow
- Implementation: separate data structure from the random generator, each unique is a full item definition
